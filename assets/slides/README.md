# 슬라이드 이미지 폴더

전자소송포털 지급명령 신청 **단계별 화면 슬라이드(총 29장)** 이미지를 이 폴더에 넣습니다.

## 넣는 방법

1. 발표자료(PPT)를 **이미지로 저장**합니다.
   - PowerPoint: `파일 → 내보내기 → 파일 형식 변경 → PNG` (모든 슬라이드 내보내기)
   - 또는 PPT를 PDF로 저장한 뒤 PDF를 페이지별 PNG로 변환
2. 파일 이름을 **정렬 순서대로** 붙입니다. 두 자리 숫자를 권장합니다.
   - `01.png`, `02.png`, `03.png` … `29.png`
   - (파일명 오름차순으로 자동 정렬되어 표시됩니다)
3. 이 `assets/slides/` 폴더에 업로드합니다.
   - GitHub 웹: 저장소 → `assets/slides` 폴더 → **Add file → Upload files** 로 29장 드래그&드롭
4. 커밋하면 `단계별 화면 안내` 페이지에 **전체가 자동으로** 표시됩니다.

## 캡션(선택)

각 슬라이드에 제목/설명을 붙이고 싶으면 `_data/slides.yml` 에
파일명과 함께 항목을 추가하세요. (없어도 이미지는 정상 표시됩니다.)

```yaml
- file: 02.png
  step: "1"
  title: "전자소송포털 로그인"
  desc: "사용자 등록한 정보로 로그인"
```

> 지원 형식: png, jpg, jpeg, webp, gif, svg
